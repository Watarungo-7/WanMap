# CLAUDE.md — WanMap プロジェクトコンテキスト

## これは何か

WanMapは、愛媛県今治市を犬連れで訪れる人のための情報サイト。スマホでURLを開けばすぐ使える。アプリのインストールもログインも不要。

「犬と今治に行ってみたい」と思った人が、このサイトを見れば必要な情報が全部わかる状態をつくる。

## 背景: いぬばり構想

今治市は年間448万人の観光客が訪れるが「住みたい街」としては選ばれていない。「犬との暮らしやすさ」を切り口に、今治を「暮らしの質で選ばれるまち」へ転換する構想。

WanMapはこの構想の初年度施策として、犬連れで今治を訪れる人に「この街、犬に本気だな」と感じてもらうためのデジタル接点。来訪者がこのサイトを見ながら街を歩き、体験をSNSでシェアしてくれることで、次の来訪者が生まれる循環を狙う。

## 今治の独自資産

- **いぬばり認証制度**: 犬同伴受入の3段階認証。Lv.1 ウェルカム（テラス席OK）/ Lv.2 フレンドリー（店内OK）/ Lv.3 ステイ（宿泊OK）
- **岡山理科大学獣医学部**: 日本に17校しかない獣医学部が今治にある。獣医学生の無料相談ブースを駅前で開催
- **しまなみ海道**: 世界的サイクリングルート。島しょ部は犬連れ散歩にも最適
- **今治タオル**: ペット用グッズとの親和性が高いブランド

## 想定ユーザーと利用シーン

- 四国・中国地方から車で今治に来る犬飼育者
- 「今治 犬連れ」「いぬばり」でGoogle検索してたどり着く
- 来訪前に「どんな店があるか」を下調べする
- 現地でスマホ片手に「次どこ行こう」と見る
- 帰宅後にSNSでシェアする際にリンクを貼る

## 技術方針

**「シンプルな情報サイト」であることが最重要。** アプリっぽい作り込みは不要。

- **フレームワーク**: Next.js (App Router) + TypeScript + Static Export
- **スタイリング**: Tailwind CSS
- **地図**: Mapbox GL JS（スポット一覧を地図上にピン表示）
- **データ**: 静的JSON。ビルド時に読み込む。CMSは不要
- **ホスティング**: Vercel（Static Export）
- **状態管理**: React state のみ。グローバル状態管理ライブラリは不要

やらないこと:

- PWA / Service Worker
- ユーザー認証・ログイン
- データベース
- サーバーサイドAPI
- localStorage への保存
- QRコード読み取り

## デザイン方針

モック（WanMap_mockup.jsx）のトーンを踏襲する。

- カラー: ダークネイビー(#1a3a4a) / ティール(#4ECDC4) / オレンジ(#FF9F43) / コーラル(#FF6B6B) / スカイ(#45B7D1)
- 認証レベル色: Lv.1=#4ECDC4 / Lv.2=#FF9F43 / Lv.3=#FF6B6B
- 背景: #f8f6f3
- フォント: Noto Sans JP (Google Fonts)
- モバイルファースト。PC表示時は中央寄せで最大幅480px程度
